## Добавить клиента

```
meridian client add alice
```

Каждый клиент получает свой уникальный ключ подключения. Команда генерирует:
- **QR-код**, отображаемый в терминале
- **HTML страницу подключения**, сохранённую локально
- **Ссылку для обмена** (если включены страницы, размещённые на сервере)

## Отобразить клиентов

```
meridian client list
```

Показывает всех клиентов с их подключениями по протоколам (Reality, XHTTP, WSS).

## Удалить клиента

```
meridian client remove alice
```

Отзывает доступ сразу. UUID клиента удаляется из всех inbound'ов на сервере.

## Несколько серверов

Используйте `--server` для выбора конкретного названного сервера:

```
meridian client add alice --server finland
```

Если у вас только один сервер, он выбирается автоматически.

## Как это работает

Имена клиентов отображаются на поля `email` 3x-ui с префиксами протокола:
- `reality-alice` — Reality inbound
- `xhttp-alice` — XHTTP inbound
- `wss-alice` — WSS inbound (режим домена)

Каждый клиент получает уникальный UUID по всем inbound'ам на сервере.
